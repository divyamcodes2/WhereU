📌 What is WhereU?

WhereU is a real-time location tracking web application that allows users to view live positions of connected users on a stunning interactive map. 🗺️
Built with 💡 Flask, 📡 Socket.IO, and 🧭 LeafletJS, it provides instant communication and location updates.

✨ Key Features

  • 🛰️ Live GPS-based tracking
  • 💬 Real-time updates via WebSockets
  • 🔒 Secure user authentication and session management
  • 🌐 Interactive and mobile-responsive UI
  • 🚪 Easy logout and account deletion options

🛠️ How to Use

  1. 📝 Sign up with a unique username and password
  2. 🔓 Log in and access the live map
  3. 📍 Your location will appear on the map automatically
  4. ⚙️ Use the sidebar to navigate, access settings, help or logout

❓ Frequently Asked Questions

Q: Is my location data saved?
A: 🚫 No! Your location is only shared live and not stored anywhere.

Q: Can I use this on mobile?
A: 📱 Absolutely! The interface is responsive and works on all screen sizes.

Q: What if I want to delete my account?
A: 🔥 Head to the options menu in the sidebar and select 'Delete Account'.
